# adaptive-ai-federation-logs

This repository is an auto-generated, privacy-safe observability archive for MCA AI Enhanced federation rounds.

## Structure
- rounds/round-NNNNNN.json (one file per round; includes full aggregated tactics snapshot)
- uploads/YYYY-MM-DD.jsonl (optional; no server IDs)
- episodes/YYYY-MM-DD.jsonl (optional; no player IDs)
- aggregates/YYYY-MM-DD.jsonl (optional)
- status/latest.json and status/history/YYYY-MM-DD.jsonl

## Privacy
- No player UUIDs, names, chat, IPs, or server identifiers are stored in GitHub logs.
